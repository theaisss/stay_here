## 📝 Описание проекта

**Stay Here** — это веб-приложение, развернутое с использованием современных практик контейнеризации и оркестрации. Проект полностью готов к продакшн-развертыванию на облачных серверах (например, AWS EC2) благодаря связке Docker и Nginx в качестве Reverse Proxy.

### Основной стек технологий:
* **Docker & Docker Compose** — контейнеризация приложения и управление зависимостями.
* **Nginx** — высокопроизводительный веб-сервер и прокси для маршрутизации трафика.
* **Ubuntu Server** — базовая операционная система для хостинга.
